## 目的

生まれ年から、その年に迎える満年齢をすぐ確認できる早見表を提供する。

入力操作を必要とせず、ページを開いた直後に主要な一覧を確認できることを最優先とする。今日時点の正確な満年齢が必要な利用者には、既存の「年齢日付計算」への導線を設ける。

## 表示範囲

- 1900年から、利用端末で取得した現在年までを表示する
- 将来年は表示しない
- 新しい年を上にした降順とする
- 1900年と現在年を含む

現在年までの行をJavaScriptで生成するため、最新年を年始に手動追加しない。開始年を変更する場合だけ、本書と実装を更新する。

## 年齢計算仕様

年齢値を静的データとして保持しない。

各行の年齢は次の式で算出する。

```text
その年に迎える満年齢 = 現在年 - 生年
```

- 現在年はブラウザの`Date`から取得する
- 誕生日を迎えたかどうかは判定しない
- 現在年生まれは0歳と表示する
- 今日時点の満年齢であるかのように表示しない
- 日付や生年月日の入力機能は追加しない

今日時点または指定日時点の正確な満年齢は、既存の「年齢日付計算」で確認できるよう案内する。

## 和暦表示仕様

西暦年だけでは、改元があった年の元号を一つに決められない。そのため、改元年は旧元号と新元号を併記する。

| 西暦 | 表示 |
| --- | --- |
| 1912年 | 明治45年／大正元年 |
| 1926年 | 大正15年／昭和元年 |
| 1989年 | 昭和64年／平成元年 |
| 2019年 | 平成31年／令和元年 |

その他の年は、次の基準で表示する。

- 1900〜1911年：明治
- 1913〜1925年：大正
- 1927〜1988年：昭和
- 1990〜2018年：平成
- 2020年以降：令和

元号の最初の年は「1年」ではなく「元年」と表示する。略号や元号合字は使用しない。

---

## 十二支表示仕様

各西暦年に対応する十二支を、次の順序で漢字一文字表示する。

```text
子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥
```

1900年を「子」とし、12年周期で算出する。

```text
十二支の位置 = (西暦年 - 1900) mod 12
```

本ページでは、日本で年の呼び方として一般的に用いられる十二支を、1月1日から12月31日までの西暦年単位で表示する。十干と組み合わせた60通りの干支や、旧暦・春節を基準にした区切りは扱わない。

## 補足情報

次の内容だけを簡潔に説明する。

- 満年齢は、生まれた日を0歳として誕生日ごとに1歳増える年齢であること
- 本ページは「その年に迎える満年齢」を表示していること
- 誕生日の前後で、今日時点の実際の満年齢が異なること
- 改元年は同じ西暦内に二つの元号があるため併記していること
- 本ページの干支欄は、一般に干支と呼ばれる十二支を西暦年単位で示していること
- 正確な年齢は「年齢日付計算」で確認できること

## FAQ

### 満年齢とは何ですか？

生まれた日を0歳とし、誕生日を迎えるたびに1歳増える年齢です。

### 数え年とは違いますか？

はい。数え年は生まれた時点を1歳とし、新年を迎えるたびに1歳増える考え方です。現在の日常生活では、一般的に満年齢が使われます。

### 誕生日前は何歳ですか？

このページは「その年に迎える満年齢」を表示しています。誕生日前の今日時点の満年齢は、一覧の数字より1歳少ない場合があります。正確な年齢は「年齢日付計算」で確認できます。

### 改元があった年の和暦はどう表示しますか？

同じ西暦年に旧元号と新元号が含まれるため、「平成31年／令和元年」のように両方を表示します。

### 毎年手動で更新されますか？

いいえ。現在年と年齢はブラウザで自動計算するため、毎年の手動更新は不要です。
