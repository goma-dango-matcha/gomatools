## 1. サービス概要

- サービス名: ゴマツール
- ブランドコンセプト: **毎日に、一歩便利。**
- サービス説明: 計算ツール、入力不要ツール、ゴマ知識、ゴマ早見表を通じて、毎日の確認や疑問解決を支える無料Webサービス。
- ブランドシリーズ: **計算ツール／入力不要ツール／ゴマ知識／ゴマ早見表**の4シリーズを中心に育成する。ゴマ早見表は、入力せず整理された一覧から必要な情報を比較・確認するシリーズ。
- ブランド育成: 単発のツール集ではなく、使いやすさ・分かりやすさ・安心感を備えた一貫性のあるサービスブランドとして育成している。
- スマホファースト: 小さな画面でも主要情報へすぐ到達でき、迷わず操作できる構成を優先する。
- タイパ重視: 短時間で答えが分かり、必要な場合だけ補足情報や関連知識へ進める体験を重視する。
- AI・Googleを意識した設計: 人にも検索エンジンにも内容と役割が伝わる見出し、文章、構造を整備する。
- 目指す方向: 利用者が毎日の疑問を迷わず解決でき、必要に応じて関連ツールやゴマ知識へ自然に移動できるサービスを目指す。

## 8. ブランド改善 共通方針

- 計算ロジック
- 日付処理

## 15. SEO状況

サイト側の確認状況:

- `robots.txt`: 全クロールを許可し、sitemap URLを指定
- `sitemap.xml`: 公開60URLを掲載。公開43ツールの漏れ・重複なし
- canonical: sitemap掲載60ページでURLと一致
- noindex: sitemap対象外のゴマ知識テンプレートだけに設定
- GitHub Pages: 正常
- Search Console: 登録済み
- ホームページ: Googleへインデックス済み
- 個別ツールページ: 順次インデックス対象
- Google Search Console所有権確認用ファイル: 設置済み
- 主要ページのtitle、meta description、OGP、構造化データ: 順次整備済み
- ゴマ知識: 公開中の全10記事へ内容に対応する参考・出典を追加済み

サイト側で実施するSEOの大きな基盤改善は完了段階。今後はSEO設定を頻繁に変更せず、Search ConsoleとAnalyticsの蓄積データ、検索結果、利用者行動を確認して判断する。

Search Console管理画面の前回確認状況（今回の作業では再確認できていない）:

- サイトマップは「取得できませんでした」が継続している。
- URL検査のライブテストでは「URL は Google に登録できます」を確認済み。
- ページはインデックス可能な状態であることを確認済み。
- インデックス登録リクエストはGoogle側のエラーで完了できていない。

2026-07-19の公開環境技術監査では、sitemap掲載60URLがすべてHTTP 200で、XML、robots、canonical、noindex、内部リンク、`/gomatools/`ベースパス、公開成果物とGitの`HEAD`の整合に重大な問題はなかった。正しいサイトマップURLは`https://goma-dango-matcha.github.io/gomatools/sitemap.xml`で、ホスト直下の`https://goma-dango-matcha.github.io/sitemap.xml`は404となる。Search Consoleの送信済み一覧で、登録された完全URLを確認する必要がある。

sitemapの`lastmod`はGit履歴上の直近の重要変更を反映していないページがあるため、次回更新時に実際の変更日と同期する。公開から日が浅いため、短期間の数値だけで大きな方針変更を行わない。

## 16. SEO運用

- Search Consoleは週1回を目安に確認する。
- URL検査は新規公開ページや重要ページを中心に行う。
